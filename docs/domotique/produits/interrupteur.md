# Interrupteurs
Un interrupteur connecté, vous permet de simuler un appui sur un interrupteur classique. Vous ne changez pas vos lampes, et vous conservez la possibilité d'appuyer manuellement sur l'interrupteur.  
Personnellement, je préfère cela à changer toutes mes ampoules pour des ampoules connectées.
